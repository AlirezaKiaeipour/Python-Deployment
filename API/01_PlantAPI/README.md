# PlantNet

**Plant identification using generated plant image**
 
  
## API

Name | Links |
------------- | ------------- |

  ## RUN
  You can run  Inference with the following command
  

  ```
  $ pip install requirements.txt
  
  python plant.py [--input_text INPUT]
  
  ```
